# themeswitcher30 README

Randomly switch themes at HH:00:00 and HH:30:00 every hour.

## Features

Randomly switch themes at HH:00:00 and HH:30:00 every hour.

You can edit the list of random themes.

```json5
{
  "themeswitcher30.themeList": [/* list of theme name */]
}
```
